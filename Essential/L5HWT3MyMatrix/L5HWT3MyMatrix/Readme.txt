Используя Visual Studio, создайте проект по шаблону Console Application.  
Требуется:  
Создать класс MyMatrix, обеспечивающий представление матрицы произвольного размера 
с возможностью изменения числа строк и столбцов.  
Написать программу, которая выводит на экран матрицу и производные от нее матрицы разных порядков. 

Решение:
	+Создать класс MyMatrix;
	+Создать класс SubMatrix;
	MyMatrix.cs:	
		+Создать конструктор класса, в параметры которого передаем количество столбцов и строк матриц;
		+Создать цикличный метод for, который будет инициализировать каждый елемент матрицы матрицой (создаем матрицу матриц):
			+Вызываем конструктор класса SubMatrix при каждой итерации ;
		+Создать метод Show, который, используя for, будет обращаться к методу вывода в SubMatrix;
			SubMatrix.cs:
				+Создать конструктор класса в параметры которого передаем количество столбцов и строк матрицы;
				+Используя цикличный метод for и метод Random заполняем матрицу случайными значениями;
				+Создать метод Show, который будет выводить матрицу на экран; 
	Program.cs:
		+Реализуем метод CheckInt, который будет проверять правильность введенных данных;
		+В теле метода Main реализуем ввод количества строк и столбцов матрицы с субматрицы;
		+Инициализируем MyMatrix и передаем введенные значения в конструктор класса;
		+Вызываем метод вывода матриц на экран.
