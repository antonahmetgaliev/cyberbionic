Задание 
Используя Visual Studio, создайте проект по шаблону Console Application.  
Требуется:  
Создать статический класс Calculator, с методами для выполнения основных арифметических операций. 
Написать программу, которая выводит на экран основные арифметические операции. 

Calculator.cs:
	+создать статический метод Sum, который будет принимать два аргумента и возвращать их сумму;
	+создать статический метод Sub, который будет принимать два аргумента и возвращать их разность;
	+создать статический метод Mul, который будет принимать два аргумента и возвращать их произведение;
	+создать статический метод Div, который будет принимать два аргумента и возвращать их частное;
Program.cs:
	+получить с клавиатуры значение a, и значение b;
	+передать значения a и b в каждый из методов и вывести на экран результат;
	+реализовать проверку деления на 0.
