Используя Visual Studio, создайте проект по шаблону Console Application.  
Требуется:  
Создать класс Article, содержащий следующие закрытые поля: 
• название товара; 
• название магазина, в котором продается товар; 
• стоимость товара в гривнах. 
Создать класс Store, содержащий закрытый массив элементов типа Article.  
Обеспечить следующие возможности: 
• вывод информации о товаре по номеру с помощью индекса; 
• вывод на экран информации о товаре, название которого введено с клавиатуры, если таких товаров нет, выдать соответствующее сообщение; 
Написать программу, вывода на экран информацию о товаре. 

Решение:
	+Создать класс Article, содержащий следующие закрытые поля: 
		+название товара; 
		+название магазина, в котором продается товар; 
		+стоимость товара в гривнах. 
	+Создать класс Store, содержащий закрытый массив элементов типа Article;
	Article.cs:
		+создать конструктор, который иницилизирует 3 вышеуказанных поля;
		+создать 3 метода, которые будут возвращать значения закрытых полей;
	Store.cs:
		+создать метод, который обеспечит добавление нового товара в магазин;
		+создать метод, который с параметров будет получать индекс и возвращать информацию о товаре, либо ошибку;
		+создать метод, который с параметров будет получать имя товара и, используя метод for, искать совпадение среди добавленных товаров;
		+создать метод, который будет возвращать массив, содержащий информацию о всех товарах;
	Program.cs:
		+спросить у пользователя сколько товаров он хочет добавить;
		+используя метод for спросить у пользователя:
			+имя товара;
			+название магазина, в котором продается этот товар;
			+цену продукта в гривнах;
		+передать данные в конструктор класса Store;
		+спросить у пользователя индекс товара, который он хочет найти;
		+передать индекс в метод поиска товаров по индексу;
		+спросить у пользователя имя товара, который он хочет найти в классе Store;
		+передать имя в метод поиска товаров по имени в классе Store;
		+вывести все товары на экранЮ используя метод Show в классе Store.
